## 权限管理平台


### 项目描述
   简易的权限管理平台，开箱即用
   
### 组件说明
    1.common 公共包
    2.domain 数据库层
    3.model 实体类包
    4.service 逻辑层
    5.web 接口层
    6.jar 外部引用层(待完成)
### 依赖组件
    1.mysql
    2.redis
    3.dwz框架
### 使用步骤
    1.建立mysql数据库
    2.执行table.sql
    3.执行 init.sql 
    4.登录页面
        http://localhost:18083
    5.默认用户名密码
        001/123456
    6.接口文档
        http://localhost:18083/swagger-ui.html
